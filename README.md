# 堆
最大（小）的完全二叉树，在这个树中，每个节点的值大于（小于）或等于其他子节点的值

## 大顶堆
最大的[完全二叉树](https://github.com/TF27674569/Tree)，满足完全二叉树，是一个有序的数据结构  
所以只需要在完全二叉树的基础上满足每个节点的值大于或等于其他子节点的值，就是大顶堆.
